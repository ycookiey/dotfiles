# lazygit キーバインド

公式キーバインド一覧: <https://github.com/jesseduffield/lazygit/blob/master/docs/keybindings/Keybindings_en.md>
lazygit内で `?` を押すと、現在のパネルのキーバインド一覧が出る。

## グローバル

| キー | 機能 |
|------|------|
| `?` | 現在のパネルのキーバインド表示 |
| `q` | 終了 |
| `Esc` | 戻る／キャンセル |
| `1`〜`5` | パネル移動 |
| `[` / `]` | タブ切替 |
| `p` / `P` | pull / push |
| `+` / `_` | パネル拡大 / 縮小 |

## Files（変更ファイル）

| キー | 機能 |
|------|------|
| `Space` | ステージ／アンステージ |
| `a` | 全ファイルをステージ |
| `Enter` | ファイル内の行・hunk単位でステージ |
| `c` | コミット（内蔵パネル） |
| `C` | 外部エディタでコミット |
| `A` | 直前のコミットにamend |
| `e` | ファイルを編集 |
| `d` | 変更を破棄 |

## Commits（コミット一覧）

| キー | 機能 |
|------|------|
| `r` | reword（内蔵パネルでメッセージ編集） |
| `R` | reword（外部エディタでメッセージ編集） |
| `e` | コミットを編集（interactive rebase） |
| `s` | 下のコミットにsquash |
| `f` | 下のコミットにfixup |
| `d` | コミット削除 |
| `Space` | そのコミットにcheckout |

## コミットメッセージ編集（内蔵パネル）

| キー | 機能 |
|------|------|
| `Tab` | summary（タイトル）↔ description（body）切替 |
| `Enter` | 確定 |
| `Ctrl+S` / `Alt+Enter` | 確定（複数行入力中でも確定） |
| `Ctrl+U` | カーソルから行頭まで削除 |
| `Ctrl+K` | カーソルから行末まで削除 |
| `Ctrl+W` | カーソル前の1単語削除 |
