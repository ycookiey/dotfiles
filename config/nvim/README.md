# Neovim キーバインド

Leader: `Space`

## カスタム（プラグイン）

### Telescope（ファジーファインダー）

| キー | 機能 |
|------|------|
| `Space f f` | ファイル検索 |
| `Space f g` | テキスト検索（live grep） |
| `Space f b` | バッファ一覧 |
| `Space f h` | ヘルプタグ検索 |

### Neo-tree（ファイルツリー）

| キー | 機能 |
|------|------|
| `Space e` | ファイルツリー トグル |

### LSP

| キー | 機能 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー情報 |
| `Space rn` | リネーム |
| `Space ca` | コードアクション |

### 補完（nvim-cmp）

| キー | 機能 |
|------|------|
| `Ctrl+Space` | 補完を手動トリガー |
| `CR` | 補完確定 |
| `Tab` | 次の候補 / スニペットジャンプ |
| `Shift+Tab` | 前の候補 |

## Vim 基本操作

### モード切替

| キー | 機能 |
|------|------|
| `i` | カーソル前に挿入 |
| `a` | カーソル後に挿入 |
| `I` | 行頭に挿入 |
| `A` | 行末に挿入 |
| `o` | 下に行挿入 |
| `O` | 上に行挿入 |
| `v` | ビジュアルモード（文字） |
| `V` | ビジュアルモード（行） |
| `Ctrl+V` | ビジュアルモード（矩形） |
| `Esc` | ノーマルモードへ |

### 移動

| キー | 機能 |
|------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w` | 次の単語先頭 |
| `b` | 前の単語先頭 |
| `e` | 単語末尾 |
| `W/B/E` | WORD単位（空白区切り） |
| `0` | 行頭 |
| `^` | 行頭（非空白） |
| `$` | 行末 |
| `gg` | ファイル先頭 |
| `G` | ファイル末尾 |
| `{` / `}` | 段落移動 |
| `%` | 対応する括弧へ |
| `Ctrl+U` | 半ページ上 |
| `Ctrl+D` | 半ページ下 |
| `Ctrl+B` | 1ページ上 |
| `Ctrl+F` | 1ページ下 |
| `H/M/L` | 画面 上/中/下 |
| `f{c}` | 行内で `{c}` へジャンプ |
| `F{c}` | 行内で `{c}` へ逆方向ジャンプ |
| `t{c}` / `T{c}` | `{c}` の手前/後ろへ |
| `;` / `,` | f/t の繰り返し/逆方向 |

### 検索

| キー | 機能 |
|------|------|
| `/{pattern}` | 前方検索 |
| `?{pattern}` | 後方検索 |
| `n` / `N` | 次/前の検索結果 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |

### 編集

| キー | 機能 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `D` | カーソルから行末まで削除 |
| `cc` | 行変更（削除+挿入） |
| `C` | カーソルから行末まで変更 |
| `yy` | 行コピー |
| `p` / `P` | 後/前に貼り付け |
| `u` | アンドゥ |
| `Ctrl+R` | リドゥ |
| `.` | 直前の操作を繰り返し |
| `J` | 下の行を結合 |
| `~` | 大文字/小文字切替 |
| `>>` / `<<` | インデント追加/削除 |

### オペレータ + モーション/テキストオブジェクト

`d`, `c`, `y` + モーション/テキストオブジェクトで範囲操作。

| キー | 機能 |
|------|------|
| `dw` | 単語削除 |
| `ciw` | 単語変更（周囲の空白を残す） |
| `di"` | `"..."` 内を削除 |
| `ci(` | `(...)` 内を変更 |
| `ya{` | `{...}` を括弧含めコピー |
| `dit` | HTMLタグ内を削除 |
| `dap` | 段落削除 |

テキストオブジェクト: `i` = inner（中身）、`a` = around（囲み含む）

### マーク・ジャンプ

| キー | 機能 |
|------|------|
| `m{a-z}` | マーク設定（ファイル内） |
| `'{a-z}` | マークへジャンプ |
| `Ctrl+O` | ジャンプリストを戻る |
| `Ctrl+I` | ジャンプリストを進む |
| `gd` | ローカル定義へ（LSP無効時） |
| `gf` | カーソル下のファイルを開く |

### ウィンドウ

| キー | 機能 |
|------|------|
| `Ctrl+W s` | 水平分割 |
| `Ctrl+W v` | 垂直分割 |
| `Ctrl+W h/j/k/l` | ウィンドウ移動 |
| `Ctrl+W q` | ウィンドウを閉じる |
| `Ctrl+W =` | ウィンドウ幅を均等化 |
| `Ctrl+W _` | 高さ最大化 |
| `Ctrl+W \|` | 幅最大化 |

### コマンドライン

| キー | 機能 |
|------|------|
| `:w` | 保存 |
| `:q` | 閉じる |
| `:wq` / `ZZ` | 保存して閉じる |
| `:q!` / `ZQ` | 保存せず閉じる |
| `:e {file}` | ファイルを開く |
| `:%s/old/new/g` | 全置換 |
| `:noh` | 検索ハイライト解除 |

### レジスタ

| キー | 機能 |
|------|------|
| `"{reg}y` | レジスタ `{reg}` にコピー |
| `"{reg}p` | レジスタ `{reg}` から貼り付け |
| `"0p` | 直前のヤンク内容を貼り付け（削除で上書きされない） |
| `"+y` / `"+p` | システムクリップボード（`unnamedplus` で自動連携済み） |
| `:reg` | レジスタ一覧表示 |
