# Yazi キーバインド

プラグイン: fzf

## ナビゲーション

| キー | 機能 |
|------|------|
| `h/j/k/l` | 親ディレクトリ/次/前/子ディレクトリ |
| `gg` | 先頭へ |
| `G` | 末尾へ |
| `Ctrl+U/D` | 半ページ上/下 |
| `Ctrl+B/F` | 1ページ上/下 |
| `H` | 前のディレクトリに戻る |
| `L` | 次のディレクトリに進む |

## ファイル操作

| キー | 機能 |
|------|------|
| `o` / `Enter` | 開く |
| `O` / `Shift+Enter` | 対話的に開く（アプリ選択） |
| `a` | 新規作成（末尾 `/` でディレクトリ） |
| `r` | リネーム |
| `y` | ヤンク（コピー） |
| `x` | ヤンク（カット） |
| `p` | 貼り付け |
| `P` | 貼り付け（上書き） |
| `Y` / `X` | ヤンク取消 |
| `d` | ゴミ箱に移動 |
| `D` | 完全削除 |

## リンク作成

| キー | 機能 |
|------|------|
| `-` | シンボリックリンク（絶対パス） |
| `_` | シンボリックリンク（相対パス） |
| `Ctrl+-` | ハードリンク |

## 選択

| キー | 機能 |
|------|------|
| `Space` | 選択トグル |
| `Ctrl+A` | 全選択 |
| `Ctrl+R` | 選択反転 |
| `v` | ビジュアルモード（選択） |
| `V` | ビジュアルモード（選択解除） |

## 検索・フィルタ

| キー | 機能 |
|------|------|
| `s` | ファイル名検索（fd） |
| `S` | ファイル内容検索（ripgrep） |
| `Ctrl+S` | 検索キャンセル |
| `z` | fzfでジャンプ |
| `Z` | zoxideでジャンプ |
| `f` | フィルタ |
| `/` | 次を検索 |
| `?` | 前を検索 |
| `n` / `N` | 次/前の検索結果 |

## パスコピー

| キー | 機能 |
|------|------|
| `cc` | ファイルパスをコピー |
| `cd` | ディレクトリパスをコピー |
| `cf` | ファイル名をコピー |
| `cn` | ファイル名（拡張子なし）をコピー |

## ソート（`,` + キー）

| キー | 機能 |
|------|------|
| `,m` / `,M` | 更新日時（昇順/降順） |
| `,b` / `,B` | 作成日時（昇順/降順） |
| `,e` / `,E` | 拡張子（昇順/降順） |
| `,a` / `,A` | アルファベット（昇順/降順） |
| `,n` / `,N` | 自然順（昇順/降順） |
| `,s` / `,S` | サイズ（昇順/降順） |
| `,r` | ランダム |

## 表示モード（`m` + キー）

| キー | 機能 |
|------|------|
| `ms` | サイズ表示 |
| `mp` | パーミッション表示 |
| `mb` | 作成日時表示 |
| `mm` | 更新日時表示 |
| `mo` | オーナー表示 |
| `mn` | 表示なし |

## ブックマーク（`g` + キー）

| キー | 機能 |
|------|------|
| `gh` | ホーム |
| `gc` | `~/.config` |
| `gd` | `~/Downloads` |
| `g Space` | 対話的にジャンプ |
| `gf` | シンボリックリンク先へ |

## タブ

| キー | 機能 |
|------|------|
| `tt` | 新規タブ |
| `tr` | タブ名変更 |
| `1`〜`9` | タブ切替 |
| `[` / `]` | 前/次のタブ |
| `{` / `}` | タブ位置を前/次と入替 |

## プレビュー

| キー | 機能 |
|------|------|
| `K` / `J` | プレビューを上/下にスクロール |
| `Tab` | ファイルをスポット表示 |

## シェル

| キー | 機能 |
|------|------|
| `;` | シェルコマンド実行 |
| `:` | シェルコマンド実行（完了まで待機） |

## その他

| キー | 機能 |
|------|------|
| `.` | 隠しファイル表示トグル |
| `w` | タスクマネージャ |
| `~` / `F1` | ヘルプ |
| `q` | 終了 |
| `Q` | 終了（cwd出力なし） |
| `Ctrl+C` | タブを閉じる（最後のタブなら終了） |
| `Ctrl+Z` | サスペンド |

## タスクマネージャ内

| キー | 機能 |
|------|------|
| `j/k` | タスク移動 |
| `Enter` | タスク詳細 |
| `x` | タスクキャンセル |
| `w` / `Esc` | 閉じる |
