# WezTerm キーバインド

## ナビゲーション

| キー | 機能 |
|------|------|
| `Ctrl+Tab` | 次へ（ペイン巡回 → 最後なら次のタブ） |
| `Ctrl+Shift+Tab` | 前へ（ペイン巡回 → 最初なら前のタブ） |
| `Ctrl+1~9` | タブ番号で直接切替 |

## タブ

| キー | 機能 |
|------|------|
| `Ctrl+T` | 新規タブ |
| `Ctrl+W` | タブを閉じる |
| `Alt+,` | タブ位置を左へ |
| `Alt+.` | タブ位置を右へ |

## タブ合成・分離

| キー | 機能 |
|------|------|
| `Alt+M` | 現在のタブの全ペインを左隣のタブに統合 |
| `Alt+Shift+M` | 現在のペインを新規タブに分離 |

## ペイン

| キー | 機能 |
|------|------|
| `Alt+D` | 縦分割 |
| `Alt+R` | 横分割 |
| `Alt+H/J/K/L` | ペイン移動（左/下/上/右） |
| `Alt+X` | ペインを閉じる |
| `Alt+Z` | ペインズーム（トグル） |
| `Alt+S` | サイズ調整モード（`H/J/K/L` で調整、`Enter` で終了） |
| `Alt+Shift+S` | ペイン幅を均等化 |

## コピー・貼り付け

| キー | 機能 |
|------|------|
| `Ctrl+C` | 選択範囲があればコピー、なければ SIGINT |
| `Ctrl+V` | 貼り付け |
| `Alt+V` | コピーモード（Vim風操作） |

### コピーモード内操作

| キー | 機能 |
|------|------|
| `h/j/k/l` | カーソル移動 |
| `w/b/e` | 単語移動 |
| `0/^/$` | 行頭/行頭(非空白)/行末 |
| `g/G` | スクロールバック先頭/末尾 |
| `H/M/L` | Viewport 上/中/下 |
| `f/F/t/T` | 文字ジャンプ |
| `;` | ジャンプ繰り返し |
| `Ctrl+B/F` | ページ上/下 |
| `Ctrl+U/D` | 半ページ上/下 |
| `v` | 文字選択 |
| `V` | 行選択 |
| `Ctrl+V` | 矩形選択 |
| `y` | コピー |
| `Enter` | コピーして終了 |
| `Escape/q/Ctrl+C` | 終了 |
